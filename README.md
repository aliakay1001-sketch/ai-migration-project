# AI & Migration Project

This project explores the use of data analysis and basic machine learning
to study migration trends and policy-related texts.

## Objectives
- Analyze migration datasets
- Apply NLP to policy and legal texts
- Explore ethical issues and bias in AI

## Tools
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
